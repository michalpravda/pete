Activity
 
run pro jiný nepovinný parametr by pouštěl "subset" toho výše: a_package_in - jednu package a_script in - jeden skript a_method_in - jednu metodu a_case_in - jeden test case.
 
Pri vyplnění a_package_in nebo a_method_in jde o "konvencni" spoustění a proto budou rovněž spuštěny všechny before/after metody/package
 
a_style_in nepovinny, pouze, pokud dle obsahu parametru ke spousteni nepujde rozhodnout, který styl uživatel myslí (budou k dipozici oba)

run 

PROCEDURE run
(
    a_suite_name_in      IN VARCHAR2 DEFAULT NULL,
    a_package_name_in    IN VARCHAR2 DEFAULT NULL,
    a_method_name_in     IN VARCHAR2 DEFAULT NULL,
    a_script_code_in     IN VARCHAR2 DEFAULT NULL,
    a_procedure_name_in  IN VARCHAR2 DEFAULT NULL,
    a_case_code_in       IN VARCHAR2 DEFAULT NULL,
    a_style_conventional IN BOOLEAN DEFAULT NULL
)

Runs test Universal run procedure. Can be used to run any unit of work of either testing style.
Other public run procedures call this one. It accepts only - configuration parameters (a_suite_name_in, a_script_code_in, a_case_code_in) or - conventional parameters (a_suite_name_in, a_package_name_in, a_method_name_in) not a combination from both sets testing style can be explicitly set by a_style_conventional_in parameter

Parameters
a_suite_name_in   Runs a suite of tests of a given name. If there are suites of both testing styles then throws an ge_ambiguous_input exception
a_package_name_in   Runs all tests following convention in a given package
a_method_name_mask_in   Runs only tests of a given mask in a given package. Must be used with param a_package_in
a_script_code_in   Runs a test script of a given code
a_case_code_in   Runs a test case of a given code
a_style_conventional   If true

Throws exceptions
ge_ambiguous_input
ge_conflicting_input
