PROCEDURE pass(a_comment_in IN VARCHAR2 DEFAULT NULL)

This assert procedure always succeeds. It's usefull to log, that something has happened if it is difficult to test it's value

fail 

PROCEDURE fail(a_comment_in IN VARCHAR2 DEFAULT NULL)

This assert procedure always fails. It's usefull in a branch of code where the program should never enter. E.g. in an exception block
